📋 Come ottenere il link corretto

  1. Vai su Google Drive e trova il file .xlsx
  2. Clicca col tasto destro → "Ottieni link"
  3. Cambia accesso a "Chiunque abbia il link" (solo visualizzazione)
  4. Copia il link e incollalo qui sotto
  5. Clicca Connetti — il dashboard si caricherà automaticamente
oppure
💡 Nota: il link Google Drive funziona solo se il dashboard è aperto tramite un server web (es. GitHub Pages, Netlify, o VS Code Live Server). Se lo apri come file locale, usa il caricamento diretto.
